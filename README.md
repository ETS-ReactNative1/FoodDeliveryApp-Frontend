# FoodDeliveryApp-Frontend
### The Frontend of a Food Delivery Mobile Application
---
# Used Concepts
- React Native
- Expo
- Google Maps API
---
# Functionalities
- Choose a category
- See all restaurants
- Add/Remove items to/from cart
- See the location of the courier and the duration of the command live, using react-native-maps
